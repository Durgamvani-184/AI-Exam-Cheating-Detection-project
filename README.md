# AI Exam Cheating Detection System

This project is a real-time AI-based system designed to detect cheating
behavior during online examinations using computer vision techniques.

## Features
- Real-time webcam monitoring
- Eye movement based cheating detection
- Multiple face detection
- Live status display using Flask web app

## Technologies Used
- Python
- OpenCV
- MediaPipe
- Flask

## Project Structure
AI-Exam-Cheating-Detection-project/
- app.py
- cheating_detector.py
- requirements.txt
- README.md

## How to Run
1. Install dependencies  
   pip install -r requirements.txt

2. Run the application  
   python app.py

3. Open browser  
   http://127.0.0.1:5000

## Use Case
This system can be used by educational institutions to monitor students
during online exams and prevent cheating.
